# Vital Market - Bosa Brasilia 🏥💊

Sistema web para la gestión y búsqueda de medicamentos en farmacias del sector de Bosa Brasilia, Bogotá.

## 📋 Descripción

Vital Market es una plataforma que permite a los usuarios:
- Consultar el inventario de medicamentos en tiempo real
- Realizar compras con carrito de compras
- Ver farmacias cercanas en el mapa
- Gestionar el stock desde un panel de control
- Generar reportes de inventario en PDF y Excel

## ✨ Características

| Módulo | Funcionalidad |
|--------|---------------|
| 🏠 **Inicio** | Búsqueda de medicamentos, productos destacados, mapa rápido |
| 💊 **Mi Botiquín** | Catálogo completo, carrito de compras, filtros por categoría |
| 🗺️ **Mapa** | Farmacias de Bosa Brasilia, filtros "Full Combo" y "Más cercanos" |
| 📊 **Panel** | Control de inventario, semaforización de existencias |
| 👑 **Premium** | Descuentos del 5%, productos destacados, beneficios exclusivos |

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Base de datos**: Firebase Realtime Database
- **Gráficas**: Chart.js
- **PWA**: Manifest para instalación como app móvil
